# Skills

Este repositório reúne documentações de skills e instruções de instalação para diferentes integrações e fluxos de IA.

## Skills disponíveis

- [AI Query Laravel](./skills/ai-query-laravel/skill.md): skill para orientar agentes a gerar consultas usando a DSL segura do pacote `eloquent-dsl/ai-query`, sem escrever Eloquent direto.

## Estrutura

Cada pasta deste diretório pode conter:

- um `skill.md` próprio com contexto da skill;
- instruções de instalação;
- exemplos de uso;
- observações sobre comandos, templates e comportamento esperado.

## Objetivo

A ideia é centralizar skills reutilizáveis para ferramentas como GitHub Copilot, Cursor, Claude ou integrações genéricas, mantendo uma documentação clara por diretório.
